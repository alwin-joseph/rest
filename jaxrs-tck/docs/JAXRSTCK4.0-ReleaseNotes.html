Jakarta EE RESTful Web Services Technology Compatibility Kit, Version 4.0
Release Notes, March 2024

Kit Contents

The Jakarta EE RESTful Web Services 4.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE RESTful Web Services TCK, signature, API, and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined in the Jakarta EE RESTful Web Services 4.0 implementation that is under test.

    • Specification tests for the testable assertions required by the Jakarta EE RESTful Web Services 4.0 Specification.

    • API tests for the public APIs and language elements defined by the Jakarta EE RESTful Web Services 4.0 Specification.

Jakarta EE RESTful Web Services 4.0 TCK Facts

The test suite bundle contains the following:

  • 2722 total tests out of which 128 tests are excluded

Platform Notes

The Jakarta EE RESTful Web Services 4.0 TCK tests and the Jakarta EE RESTful Web Services 4.0 Compatible Implementation (CI) can be run with Java Platform, Standard Edition version 17.0 (Java SE 17). See Compatible Runtime for Jakarta EE RESTful Web Services 4.0 for additional information.

Refer to the Technology Compatibility Kit User's Guide for Jakarta EE RESTful Web Services (HTML, PDF) for platform compatibility requirements and notes.

Compatible Runtime for Jakarta EE RESTful Web Services 4.0

A Compatible Runtime for compatibility testing of implementations based upon the Jakarta EE RESTful Web Services 4.0 Specification is as follows:

  • Java SE 17 which must be used with a Jakarta EE profile/application server, such as the GlassFish 8.0.0+ (with Java SE 17+)

  • A Compatible Implementation (CI) of Jakarta EE RESTful Web Services 4.0 is available as part of the Eclipse GlassFish 8.0.0+ full profile, or the Eclipse GlassFish 8.0.0+, Web Profile

The Jakarta EE RESTful Web Services 4.0 TCK tests have been run on the following platforms:

  • CentOS Linux 7
